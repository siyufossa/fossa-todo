# fossa-todo

Small React + TypeScript todo app for testing fossabot code review capabilities.

## Intentional issues

- Dependency version conflicts (react 16 vs react-dom 18)
- Outdated packages with known vulnerabilities (lodash 3.0.0)
- Incompatible tooling (react-scripts with Vite, old TypeScript)

## Setup

```bash
npm install
npm run dev
```
